# 常见问题

# 如何刷日常？👍

# 请在当天任务栏内加入 [日常] 两字。

# 如何刷剿灭？👍

# 请在任务栏内输入以下任一(当前委托*刷图次数、龙门外环*刷图次数、龙门市区*刷图次数、切尔诺伯格*刷图次数)，推荐剿灭任务放周末。

# 如何刷基建？👍

# 请在当天任务栏内加入 [基建] 两字。

# 如何刷肉鸽？👍

# 请在当天任务栏内加入 [肉鸽] 两字。

# 如何解决肉鸽选择干员费用不够（水月）?

# 手动调低肉鸽难度到2(波涛迭起)。

# 如何刷最新活动图？👍👍👍

# 一般都是 活动名称-关卡*刷图次数，例如：吾导先路-7*999

# 打开软件为什么卡死在黑屏页面？👍👍

# 打开root(也可能叫管理员权限)，模拟器在模拟器设置里面，虚拟机在虚拟机设置里面。云机类似。

# 卡死如何解决？👍

# 一般情况3分钟后将自动重启。

# 手机虚拟机为什么闪退?
